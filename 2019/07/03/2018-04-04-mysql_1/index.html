sql及其规范

  1. 在数据库系统中，SQL语句不区分大小写(建议用大写) 。但字符串常量区分大小写。建议命令大写，数据库名称、表名称、字段名称全部小写，用反引号括起来；

  2. SQL语句可单行或多行书写，以“;”结尾。关键词不能跨多行或简写。

  3. 用空格和缩进来提高语句的可读性。子句通常位于独立行，便于编辑，提高可读性。

  4. 单行注释：-- 多行注释：/*......*/
